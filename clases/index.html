Clases

Bitácora digital de Franco Castañeda Becerra, estudiante de la Escuela de Arquitectura y Diseño de la PUCV, para el curso de Diseño desde el código, segundo semestre 2021.

Clase 1: Introducción al diseño front-end

Clase 13 de Agosto

Vista introductoria a los Objetivos del curso, cómo funciona un sitio web, lenguajes: HTML, CSS, JS, arquitecturas básicas y metodologías, herramientas de trabajo, además de aspectos sobre el curso: Trabajos y evaluaciones

Clase 2: HTML

Clase 20 de Agosto

Se enseña cómo escribir HTML: etiquetas,atributos y contenido, anidación e indentación. Recomendaciones de escritura y Recomendaciones para nombres de archivos y carpetas. Estructura básica, semántica y etiquetas de secciones; etiquetas semánticamente neutras; imágenes y multimedia; hipervínculos. Cómo referenciar archivos, y Validación de HTML.

Clase 3: CSS

Clase 27 de Agosto

Objetivos de la clase: ¿Cómo funcionan los hipervínculos? Introducción a CSS, cómo escribir CSS. Selectores, propiedades y valores. Declaraciones y reglas, recomendaciones de escritura ¿Cómo asociar una hoja de estilos a un documento HTML? Selectores CSS. Elementales: etiqueta, clase, id. Selectores por atributos, pseudo-selectores, pseudo-elementos, especificidad, herencia y recursos de apoyo.

Clase 4: Sesión Práctica

Clase 03 de Septiembre

En esta sesión se desarrolla un ejercicio práctico que ejemplifica el trabajo en la bitácora personal, aplicando los conceptos abordados en clases anteriores. Se introduce también el uso de flex para diagramación.

Clase 5: Tipografía; Modelo de caja

Clase 10 de Septiembre

Se presentaron:
  • Variables CSS
  • Tipografía para la web: Declaración de una tipografía, Propiedades generales, @font-face, Usos prácticos: Cargar fuentes de Google Fonts, descargadas y otras.
  • Introducción al uso de color
  • Introducción al Modelo de caja
  • Recursos de apoyo

Clase 6: Técnicas de Diagramación: Flex

Clase 24 de Septiembre

Se presentaron: Posicionamiento absoluto/relativo, diagramación con Flexbox. Dirección, ejes y orden. Distribución y dimensiones.

Clase 7: Técnicas de Diagramación: Grid

Clase 01 de Octubre

Introducción a Grid CSS, vocabulario básico: contenedor, líneas, celdas, tracks y áreas. Trabajo con unidades y revisión de unidad “fr”. Creación de grillas: grid-template-columns y grid-template-rows. Espaciado: grid-gap. Revisión de ejemplos y recursos de apoyo.

Clase 8: Ejercicios de Diagramación + JS básico

Clase 08 de Octubre

En esta clase se desarrolla un ejercicio práctico para ejemplificar el uso de Flex y Grid. También se ilustra el uso de media queries para la construcción de diseño responsivo a partir de técnica mobile first. También se introducen los principios de javascript a partir de algunos ejemplos.

Clase 9: JS y preprocesadores

Clase 15 de Octubre

  • Preprocesadores CSS: Introducción, Métodos de uso, SCSS: Anidación de declaraciones, SCSS: Uso de variables, SCSS: Operaciones, Funciones y Mixins, SCSS: Uso de import.
  • Manejo de dependencias: Node.js + NPM, Línea de comando.

Clase 10: Accesibilidad y SEO

Clase 22 de Octubre

  • Accesibilidad web: qué comprende el concepto de accesibilidad
  • Por qué es importante la accesibilidad
  • Principios de accesibilidad web: perceptible; operable; comprensible; robusto
  • Niveles de conformidad
  • Técnicas básicas para la accesibilidad
  • Validación de accesibilidad
  • SEO. Cómo funciona un buscador
  • Tećnicas básicas de SEO: robots.txt; metadatos; estructura de títulos; sitemap.xml; URL canónica; idioma; URLs amigables; otros factores
  • Metadatos Opengraph

Clase 11: Revisión proyectos grupales

Clase 29 de Octubre

Clase de materia y revisión final, cada grupo presentó sus avances respecto al proyecto final, para poder obtener retroalimentaciones y correcciones en tiempo real con los profesores. En adición se pudieron ver ejemplos de ciertos componentes para poder aplicar en el código.
